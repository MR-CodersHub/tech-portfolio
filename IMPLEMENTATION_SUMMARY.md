## 🚀 How to Use

### 1. Open the Project
Simply open `index.html` in a modern web browser.

### 2. Navigate the Site
Use the navigation bar to explore all pages.

### 3. Test Authentication
**User Flow:**
- Click "Sign Up" (will be in profile dropdown when implemented)
- Or navigate to `/auth/user/signup.html`
- Create an account
- You'll be auto-logged in and redirected to user dashboard

**Admin Flow:**
- Navigate to `/auth/admin/admin-login.html`
- Email: `admin@gmail.com`
- Password: `admin123`
- Access admin dashboard

### 4. Test Blog
- Navigate to `/public/pages/blog.html`
- Use search bar to search articles
- Click category filters
- Click "Read More" to view full article

### 5. Test Contact Form
- Navigate to `/public/pages/contact.html`
- Fill out the form
- Submit to see validation in action
